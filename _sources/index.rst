NSI - SNT
==========

La spécialité NSI, acronyme de Numérique Sciences Informatiques, regroupe des enseignements qui abordent différents aspects de l'informatique:

-  L'architecture matérielle
-  La gestion des données
-  La communication en réseau
-  La programmation en langage Python
-  Les interfaces homme machine
